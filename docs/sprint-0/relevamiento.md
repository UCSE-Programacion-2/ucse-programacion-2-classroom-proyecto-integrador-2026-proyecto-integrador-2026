# Sprint 0 - Relevamiento

## 1. Resumen del negocio

La vie en Rose es una cafetería ubicada en la provincia de Jujuy, dedicada al rubro gastronómico. El local ofrece una amplia variedad de productos como cafés, infusiones, tortas, postres, sándwiches y bebidas frías.

El público objetivo está compuesto por personas que buscan consumir en el local o consultar productos mediante redes sociales. Actualmente, el negocio utiliza principalmente Instagram como medio de comunicación con los clientes y para mostrar sus productos.

Ubicación del local:

Red social:
https://www.instagram.com/lavieenrosejujuy/?hl=es%2F

---

## 2. Problemas o necesidades detectadas

- No existe un catálogo digital organizado en una plataforma propia.
- Los clientes deben consultar precios y disponibilidad a través de redes sociales.
- No hay un control digital del stock de productos.
- La actualización de precios y productos se realiza de forma manual.
- No existe una plataforma que permita visualizar el menú de forma clara y estructurada.

---

## 3. Características del negocio que impactan en la aplicación

El sistema a desarrollar deberá permitir:

- Organizar los productos en categorías:
  - Cafés
  - Tortas y postres
  - Bebidas frías
  - Sándwiches
  - Combos

- Mostrar información detallada de cada producto:
  - Nombre
  - Precio
  - Descripción
  - Imagen
  - Categoría
  - Stock disponible

- Implementar un sistema de gestión (CRUD) que permita:
  - Agregar productos
  - Modificar precios
  - Actualizar stock
  - Eliminar productos discontinuados

---

## 4. Evidencia visual

Las imágenes del negocio se encuentran en la carpeta `fotos/`.

Se incluirán imágenes representativas de:

- Fachada del local
- Zona de atención
- Productos (cafés, tortas, bebidas, sándwiches)
- Menú o cartelería
- Publicaciones de redes sociales

Estas imágenes permitirán comprender el estilo visual del negocio y servirán como base para el diseño del e-commerce.

---

## 5. Logo

El logo del comercio se encuentra en la carpeta `logo/`.

El diseño presenta una estética elegante con predominio de tonos rosados y una tipografía cursiva, transmitiendo una identidad visual delicada y acorde al concepto de cafetería.

---

## 6. Decisiones iniciales para el Sprint 1

A partir del relevamiento realizado, se definieron las siguientes acciones para el próximo sprint:

- Diseñar la página principal del e-commerce
- Crear la estructura del catálogo de productos
- Definir categorías y atributos
- Cargar productos reales
- Iniciar el diseño del panel administrador
