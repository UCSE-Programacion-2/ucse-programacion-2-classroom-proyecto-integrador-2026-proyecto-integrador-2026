## 🏢 Cliente Real Elegido

**Nombre del Negocio:** Idalia, accesorios con estilo

**Rubro:** Papelería, regalería y Accesorios

**Justificación:** El negocio cuenta con un catálogo de productos que puede organizarse en distintas categorías, como papelería, accesorios y artículos de regalería. La variedad de productos y atributos, como precio, stock, descripción, imagen, color y categoría, justifica la implementación de una base de datos y operaciones CRUD para gestionar la información del emprendimiento. Actualmente, el manejo manual del movimiento de productos puede dificultar la actualización de stock, precios y disponibilidad, por lo que un e-commerce permitiría digitalizar estos procesos y mejorar la administración del catálogo.
